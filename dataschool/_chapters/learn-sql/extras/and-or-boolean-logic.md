The order of operations for an AND and OR Statements explained in Boolean Logic

AND:
1 AND 1 = 1 <=> 1 * 1 = 1
0 AND 1 = 0 <=> 0 * 1 = 0
etc.

OR:
0 OR 0 = 0 <=> 0 + 0 = 0
0 OR 1 = 1 <=> 0 + 1 = 1
etc.

The only exception is
1 OR 1 = 1 <=> 1 + 1 = 1
